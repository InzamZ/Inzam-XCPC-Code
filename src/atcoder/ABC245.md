# AtCoder Beginner Contest 245

## **A - Good morning**

### 题目大意

给两个时间，包含小时和分钟，问哪个早一点。相同算前者早。

### 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

#define FREAD 1

const int maxn = 1e5 + 10;
const int maxb = 110;
const int inf = 1e9;
const ll Inf = 1e18;
const ll mod = 1e9 + 7;

#ifdef FREAD
inline char nc()
{
    static char buf[100000], *p1 = buf, *p2 = buf;
    return p1 == p2 && (p2 = (p1 = buf) + fread(buf, 1, 100000, stdin), p1 == p2) ? EOF : *p1++;
}
template<typename T>
inline T read(T &x)
{
    x = 0;
    T w = 1;
    char ch = 0;
    while (ch < '0' || ch > '9') {  // ch 不是数字时
        if (ch == '-')
            w = -1;        // 判断是否为负
        ch = nc();               // 继续读入
    }
    while (ch >= '0' && ch <= '9') {  // ch 是数字时
        x = x * 10 + (ch - '0');  // 将新读入的数字’加’在 x 的后面
        // x 是 int 类型，char 类型的 ch 和 ’0’ 会被自动转为其对应的
        // ASCII 码，相当于将 ch 转化为对应数字
        // 此处也可以使用 (x<<3)+(x<<1) 的写法来代替 x*10
        ch = nc();  // 继续读入
    }
    x *= w;
    return x;  // 数字 * 正负号 = 实际数值
}
#endif

int a,b,c,d;

int main()
{
    // FIO;
    // read(T);
    while (T--) {
        ans = 0;
        read(a);read(b);read(c);read(d);
        if (a==c)
        {
            if (b<=d)
                ans = 1;
            else
                ans = 2;
        }
        else 
        {
            if (a<=c)
                ans = 1;
            else
                ans = 2;
        }
        if (ans==1) printf("Takahashi\n");
        else printf("Aoki\n");
    }
    return 0;
}
```

## **B - Mex**

### 题目大意

给N个整数，找到最小的没有出现的非负整数。($A_i \leq 2000$)

### 解题思路

由于数据范围很小，直接记录每个数是否出现，最后遍历找答案。

## **C - Choose Elements**

### 题目大意

给定两组数，其中每组N（$N \leq 2e5$）个数，每个位置选择两组数的一个，问是否可以组成一组数满足相邻的绝对值之差不超过 K 。

### 解题思路

每个选项都是无后效性的，就是你现在选哪一个只跟相邻的有关，所以采用DP思路。每次判断这个位置的数可不可以被选择到，判断依据是前一个位置中可以被选择的数，是否与当前的数绝对值之差在范围之内。最后答案就是最后一个位置是否至少有一个数可以被选择，时间复杂度是$O(N)$

## **E - Wrapping Chocolate**

### 题目大意

给你N个巧克力和M个盒子，有长宽限制。巧克力和盒子不可以旋转，巧克力只能放在刚好或者比它大的盒子里面。问是否能够找到一种方案将巧克力全部装上。

### 解题思路

我一开始是先找满足一条边的，然后遍历去找另一条边满足的，但这样相当于固定了一条边，找不到还是需要改变这条边继续找。正确的思路是先按照一条边排序，然后按从大到小顺序遍历，遇到盒子就把另一条边放入一个队列，遇到巧克力就在当前队列里找到一个最小的且大于巧克力的另一条边的盒子，然后删除。这样看似只是处理了没有排序的边，但由于排序后从大到小遍历，隐含满足了排序的边一定会满足的条件。

还有在排序的时候，在排序的边相同的时候，盒子要在巧克力前面（相对的）。总之就是要让同样宽的盒子和巧克力在遍历的时候，盒子先遍历，这样盒子就可以优先加入队列中。维护队列可以使用`multiset`，时间复杂度是$(M+N)log_2{(M+N)}$。

### 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

#define pii pair<int,int>
#define pll pair<ll,ll>
#define REG register
#define FREAD 1

const int maxn = 2e5 + 10;

#ifdef FREAD
inline char nc()
{
    static char buf[100000], *p1 = buf, *p2 = buf;
    return p1 == p2 && (p2 = (p1 = buf) + fread(buf, 1, 100000, stdin), p1 == p2) ? EOF : *p1++;
}
template<typename T>
inline T read(T &x)
{
    x = 0;
    T w = 1;
    char ch = 0;
    while (ch < '0' || ch > '9') {  // ch 不是数字时
        if (ch == '-')
            w = -1;        // 判断是否为负
        ch = nc();               // 继续读入
    }
    while (ch >= '0' && ch <= '9') {  // ch 是数字时
        x = x * 10 + (ch - '0');  // 将新读入的数字’加’在 x 的后面
        // x 是 int 类型，char 类型的 ch 和 ’0’ 会被自动转为其对应的
        // ASCII 码，相当于将 ch 转化为对应数字
        // 此处也可以使用 (x<<3)+(x<<1) 的写法来代替 x*10
        ch = nc();  // 继续读入
    }
    x *= w;
    return x;  // 数字 * 正负号 = 实际数值
}
#endif
int T = 1, n, ans = 0, m;

multiset<int>S;

struct inode {
    bool type;
    int w, h;
    bool operator < (const inode &rhs) const
    {
        if (w == rhs.w) {
            if(type  == rhs.type)
                return h < rhs.h;
            return type < rhs.type;
        }
        return w < rhs.w;
    }
};

vector<inode> v;

int main()
{
    // FIO;
    // read(T);
    while (T--) {
        ans = 1;
        read(n); read(m);
        v.resize(n + m);
        for (int i = 0; i < n; ++i) {
            v[i].type = 0;
            read(v[i].w);
        }
        for (int i = 0; i < n; ++i) {
            v[i].type = 0;
            read(v[i].h);
        }
        for (int i = n; i < n + m; ++i) {
            v[i].type = 1;
            read(v[i].w);
        }
        for (int i = n; i < n + m; ++i) {
            v[i].type = 1;
            read(v[i].h);
        }
        sort(v.begin(), v.end());
        for (int i = n + m - 1; ans && i >= 0; --i) {
            if (v[i].type)
                S.insert(v[i].h);
            else {
                auto tmp = S.lower_bound(v[i].h);
                if (tmp != S.end())
                    S.erase(tmp);
                else
                    ans = 0;
            }
        }
        if (ans)
            printf("Yes\n");
        else
            printf("No\n");
    }
    return 0;
}
```
